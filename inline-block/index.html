CSS

element樣式

block element

這個element是會佔用整個整行的空間(也就是滿版)，即便你的資料沒有這麼多。
舉例：<div> <h1> <p>
不過可以透過設定寬度高度去控制樣式。

inline element

依照inline element的內容去調整整個element的寬度，
舉例：<a> <br> <input>
但是無法修改寬度與高度，即便加入margin與padding。

inline-block

可以在display中設定inline-block
display:inline-block;
這樣就可以讓block不佔用滿版，但還可以控制其寬度與高度。
